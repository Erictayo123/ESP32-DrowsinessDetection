# ESP32-DrowsinessDetection
Drowsiness detection using ESP32-Cam and processed on local server such as Laptop or Desktop

--What You need is--
1. ESP32-CAM in my case im using ESP32S3-CAM
2. ESP8266 to be output controller
3. Buzzer and Motor as output of the drowsiness
4. Any wifi router
5. Laptop or Desktop for Image Processing

--How to Use--
1. Set up the ESP32 by uploading the Code from Arduino IDE and change the code in the "// Enter your WiFi credentials" as your router ssid and password, then upload.
2. After that IDE log will show your ESP32 ip address, copy and paste that to your browser. Make sure your device(laptop or desktop) is connected to the same wifi as the ESP32
